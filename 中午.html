事件冒泡？阻止浏览器默认行为？对应的原生兼容性写法该怎么写？

冒泡：在 冒泡 中，内部元素的事件会先被触发，然后再触发外部元素
兼容 function stopPropagation(e) {        e = e || window.event;        if(e.stopPropagation) {  //Firefox/Chrome/Opera/Safari             e.stopPropagation();        } else {            e.cancelBubble = true;  //ie        }   }  

阻止浏览器默认行为？对应的原生兼容性写法该怎么写？

兼容处理： function stopPropagation(e) {       e = e || window.event;       if(e.preventDefault) {  //Firefox/Chrome/Opera/Safari            e.preventDefault();       } else {           e.returnValue = false;  //ie       }   }  

ajax原理

1．创建XMLHttpRequest实例对象。2．使用open（）和send（）方法向服务器发送资源请求。3. 使用responseText或responseXML属性，获取服务器返回的数据。4. 使用onreadystatechange函数监听服务器的响应

描述jsonp

动态创建JS标签，向后端发送请求，返回回调函数的执行，以数据作为实参传入的一段JS脚本。
